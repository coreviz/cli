# @coreviz/cli

CoreViz CLI tool - A command-line interface for CoreViz.

## Installation

```bash
npm install -g @coreviz/cli
```

## Usage

```bash
# Run directly with npx
npx @coreviz/cli

# Or if installed globally
coreviz
```
